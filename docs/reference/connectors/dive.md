# DIVE Connector

Exports Choregraph pipeline results to DIVE-compatible VisuSpec XML format. Translates
cached data and field metadata into the XML structure consumed by the DIVE C++ visualization
kernel. Handles metadata extraction, field statistics, and XML generation/merging with
existing specification files.

::: choregraph.connectors.dive
    options:
      members_order: source
      show_source: true
