# Choregraph (Facade)

The main entry point for pipeline lifecycle management. Orchestrates XML spec parsing,
Kedro project generation, pipeline execution, data caching, and DIVE VisuSpec export.

This is the only class exported from the `choregraph` package — all interaction starts here.

::: choregraph.choregraph
    options:
      members_order: source
      show_source: true
